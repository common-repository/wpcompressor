== Description ==
= Compress for WordPress =
Since WordPress 2.5 are posts not gzip compressed. *wpCompressor* automatically compresses the data output and boosts the speed performance of your blog.


= Features =
* Gzip data compress
* Very fast code execution
* No user interface required: Install, done!

== Installation ==
1. Download the *wpCompressor* plugin
1. Unzip the archive
1. Upload the folder *wpcompressor* into *../wp-content/plugins/*
1. Go to tab *Plugins*
1. Activate *wpCompressor*
1. Ready
